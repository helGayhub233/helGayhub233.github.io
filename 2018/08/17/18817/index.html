JavaScript ( 简称：”JS” )， 是一种小型、轻量级、面向对象、跨平台的客户端脚本语言。已经被广泛用于 Web 应用开发，常用来为网页添加各式各样的动态功能，为用户提供更流畅美观的浏览效果。通常 JavaScript 脚本是通过嵌入在HTML中来实现自身的功能的。

JavaScript 程序可以在多种平台下运行，如：windows、linux、mac、ios等，且脚本程序只能在客户端的浏览器来运行，不能在服务器端来运行。

JavaScript 能干什么？

  • 表单验证：是JS最基本的功能(如下图)
  • 动态HTML：可以实现一些动态的、重复的效果
  • 交互式：人机交互，通过键盘或鼠标，与网页中的元素进行交互。



简介

名称的由来：最初是 网景公司(Netscape) 开发的 “LiveScript”，为自己的浏览器Navigator2.0开的客户端语言。想借助 Java 的名气很快成长起来，因此改名为 “JavaScript”。

Java 和 JavaScript 没有直接的联系。

变量

变量，是计算机语言中能储存计算结果或能表示值抽象概念。用于让你能够把程序中准备使用的每一段数据都赋给一个简短、易于记忆的名字，因此它们十分有用。


概念

变量本身是没有类型的，变量的类型来自于变量的值。

基本概念：

  • 变量是变化的一个量
  • 变量可以看成是一个”未知数”，x=10
  • 变量可以看成是一个”符号”、”代号”
  • 变量可以形象的看成是宾馆的”房间号”
  • 变量一般是指程序的数据
  • 变量是在内存中临时存在和运行的数据
  • 内存可以看成是一个个”小格子”，每个格子可以储存一个变量的名称和值

规则

命名规则：

  • 变量名可以包含字母、数字、下划线
  • 变量的名称一定要有意义
  • 变量名不能以数字开头，可以以字母或下划线开头。如：var 2name;(错误写法)
  • 变量名不能是系统关键字，如：var、switch、for、try、case、else、while等。
  • JavaScript 中的变量名是区分大小写的，如：name 和 Name 是两个变量
  • JavaScript 中每条语句一般以英文下的分号(;)结束，这个分号不是必须的，为了向 PHP 兼容最好加上分号
  • 如果变量名由多个单词构成的话，该如何表示？
    • “驼峰式命名”：第一个单词全小写，后面每一个单词首字母大写，如：var getUserName
    • “下划线命名”：所有单词全小写，中间用下划线连接，如：var get_user_name

类型

变量的数据类型，是指这个类型来源于”变量的值”，换句话说：值是什么类型，变量就是什么类型。

数据类型：数值型、字符型、布尔型、未定义型(undefined)、空型(null)、数组(array)、对象(object)、函数(function)

变量的数据类型，分两大类：

  • 基本数据类型：数值型、字符型、布尔型、未定义型、空型(变量名只能存一个值)
  • 引用数据类型：数组、对象、函数(变量名可能存多个值)

条件判断、循环语句

条件语句是用来判断给定的条件是否满足(表达式值是否为0)，并根据判断的结果(真或假)决定执行的语句，选择结构就是用条件语句来实现的。条件语句可以给定一个判断条件，并在程序执行过程中判断该条件是否成立，根据判断结果执行不同的操作。

if 条件判断

描述：条件成立，执行成立(true)下的代码，反之执行不成立(fasle)下的代码。

结构一：只判断真，条件为假时什么也不做

结构二：既判断真，也判断假

结构三：多条件判断

注意：虽然有多个条件，但各条件之间是”或”的关系。而且只能有一个条件成立，不能同时满足多个条件。


switch 分支语句

描述：根据一个变量的不同取值，来执行不同的代码。

结构说明：

  • switch、case、break、default都是系统关键字，都必须全小写
  • switch 后的小括号()：小括号内一般是一个变量名，这个变量可能有不同的取值
  • 每个 case 的值，与变量的值进行比对，如果一致就执行该 case 后的代码
  • 所有的 case 都是”或”的关系，只能有一个 case 会满足条件
  • 每个 case 中的代码执行完毕后，必须要用 break 语句结束，结束后程序将跳到 switch 结束大括号之后运行
  • 如果不写 break 语句，下边所有 case 语句都会执行一遍

函数

函数，是将一段公共的代码进行封装，给它起个名字叫”函数”。它的作用是一次定义，多次调用，省去了代码的重写。例如：用户名的验证，验证码验证、邮箱验证、手机号码验证等. . .

例如：判断两个数的大小，只需封装代码结构之后调用

定义格式：


说明

  • function：为系统关键字，全小写
  • 函数名称：创建函数时需要定义一个名称，命名规则与变量命名一样
  • ( )：是定义函数时需要接收数据的参数，按具体情况分析是否有参(可有可无)
  • 在参数中，不需要声明，直接使用
  • 在参数中，形参的名称与变量的命名规则一样。且形参不能用 “var” 关键字定义
    • 型参(形式参数)：参数里用形象的概念表示的参数，作用是来接收调用者传过来的数据，与变量相似
    • 实参(实际参数)：参数里实实在在的数据值，传递过来的真实数据
  • 函数功能代码：定义函数名称后紧跟着的是{ }中的函数代码
  • return：用于向函数调用者返回一个值，并立即结束函数的运行
    • return 用于终止函数的运行
    • break 用于终止各种循环的运行

变量

在函数中，变量分为：全局变量、局部变量。

  • 全局变量：可以在网页的任何地方(函数内、外部)使用的变量，称为”全局变量”
    • 在函数外部定义的变量，就是”全局变量”
    • “全局变量”既可以在函数外使用，也可以在函数内部使用
    • “全局变量”在网页关闭时，自动消失(释放空间)
  • 局部变量：只能在函数内部使用的变量，称为”局部变量”
    • “局部变量”在函数内部定义，在函数内部使用，则在函数外无法使用
    • “局部变量”在函数执行完毕就消失了

注意事项：

  • 在函数内部，省略关键字 var 定义的变量为 “全局变量”
  • var 关键字尽量别省略，省略后 “全局变量” 和 “局部变量” 无法区分

拷贝传值：将一个变量的值”拷贝”一份，传给了另一个变量。两者没有任何联系，起到复制作用。
拷贝传值中，两个变量之间没有任何联系，修改其中一个变量的值，另一个不会改变。

基本数据类型存在于”栈内存”中，因为基本数据类型只有一个变量名和变量值。

引用传值：将一个变量的数据地址”拷贝”一份，传给了另一个变量。两者是有联系的，起到
引用传值中，两个变量指向”同一个地址”，大家共享同一份数据，如果其中一个变量值发生改变则另一个变量值也发生改变。

引用数据类型存在于”栈内存”中，因为引用数据类型有多个值，将地址传入参数中。

引用数据类型的存储分两步：

  • 变量名和内存地址存在”快速内存” “栈内存” 中
  • 数据存在”满速内存” “堆内存” 中

DOM 模型

DOM，即：(Document Objcet Model)文档对象模型，主要提供了访问和操作HTML标记的方式。官方定义中 DOM 可以使用脚本，动态的访问或操作网页的内容、网页外观、网页结构。

DOM包含：图片标记、表格标记、表单标记、body、html标记等. . .

DOM 分类

  • 核心 DOM：提供了同时操作 HTML 文档 和 XML文档 的公共属性和方法
  • HTML DOM：针对 HTML 文档提供的专用的属性方法
  • XML DOM：针对 XML 文档提供的专业的属性和方法
  • CSS DOM：通过了操作 CSS 的属性和方法
  • Event DOM：事件对象模型，如：onclick、onload等. . .

DOM 模型被构造为对象的树，称为节点树（就是HTML各个元素构成的树状结构）：


节点关系

即节点数中每一个节点之间的相互联系。

  • 根节点：一个 HTML 文档只有一个根，它就是 HTML 节点
  • 子节点：某一个父节点的下级节点
  • 父节点：某一个节点的上级节点
  • 兄弟节点：两个子节点同属于一个父节点


节点类型

即节点和节点之间关系的类型。

  • document 文档节点：代表整个网页，不代表任何 HTML 标记，但它是 html 节点的父节点
  • element 元素节点：指任何 HTML 标记，每一个 HTML标记就称一个”元素节点”，它拥有文本节点和属性节点
  • attribute 属性节点：指 HTML 标记的属性
  • text 文本节点：是节点树的最底层节点


核心 DOM 属性、方法

节点访问

当访问核心 DOM 中某个节点时，都是从根节点开始的(html节点)。

注意：当 body 属性出现时才能进行操作。

  • nodeName：节点名称
  • nodeValue：节点的值，即节点中的内容，元素节点没有值
  • firstChild：第 1 个子节点
  • lastChild：最后 1 个子节点
  • childNodes：子节点列表，是一个数组(索引)
  • parentNode：父节点

节点属性

对节点的属性进行操作。

  • seAttribute(name)：获取某个节点属性的值
  • setAttribute(name,value)：给某个节点添加一个属性
  • removeAttribute(name)：删除某个节点的属性

节点创建

  • createElement(tagName)：创建一个指定的 HTML 标记
    • tagName：是指不带尖括号的 HTML 标记名称
    • 举例：var imgObj = document.createElement(“img”);
  • appendChild(elemntObj)：将创建的节点，追加到某个父节点下
    • elementObj：是创建的一个子节点
    • 举例：document.body.appedChild(imgObj);
  • removeChild(elemntObj)：删除子节点


HTML DOM 属性、方法

在核心 DOM 中，网页中节点层级很深时，访问这个节点时将十分麻烦，所以， HTML DOM 中就提供了更为便捷的方式，通过 id 直接找到节点方法，而不再从 HTML 根节点开始。

特性

  • 每一个 HTML 标记，都对应一个元素对象。如：<img>对应一个图片对象
  • 每一个 HTML 标记属性，与对应的元素对象的属性，一一对应
  • 核心DOM中的属性方法，元素对象都能用

HTML DOM 方法

  • getElementById()：查找网页中指定 id 元素的对象，并将其返回
    • 语法：var obj = document.getElementById(id);
    • 参数：id 是网页中标记属性的值
  • getElementsByTagName()：查找网页中指定的 HTML 标记，并将其返回数组
    • 语法：var arrObj = parentNoe.getElementsByTagName(tagName);
    • 参数：tarName 为要查找的标记名称(不加尖括号)

HTML DOM 属性

  • tagName：标签名称，与核心 DOM 中 nodeName 一样
  • className：CSS 类的样式
  • id：同 HTML 标记中 id 属性相同
  • title：同 HTML 标记中 title 属性相同
  • style：同 HTML 标记中 style 属性相同
  • innerHTML：添加 HTML 标记中的所有内容，包含 HTML 标记等
  • offsetWidth：元素对象的宽度，不带单位
  • offsetHeight：元素对象的高度，不带单位
  • scrollWidth：元素对象的总宽度，包括滚动条中的内容，不带单位
  • scrollHeight：元素对象的总高度，包括滚动条中的内容，不带单位
  • scrollTop：指内容向上滚动上去了多少距离(有滚动条时才有效)
  • scrollLeft：指内容向左滚动过去了多少距离(有滚动条时才有效)

  • 通过 name 属性来获取表单元素对象：表单中所有元素的起点都必须是 document 对象
    • 语法：document.formObj.elementObj;
    • 举例：document.form1.username.value.length;

事件返回值
事件的返回值，会影响对象的默认动作，如果事件返回 false，则阻止默认动作的执行;如果事件返回 true 或空，则默认动作继续执行。

input 对象

一个<input>标记，就是一个 input 对象。

input 对象属性(以 type=text 为例)

  • name：表单元素的名称
  • value：表单元素的值，用户输入的内容可以通过该属性来获取
  • size：表单长度
  • maxlength：表单元素的最大长度(最多可输的字符数)
  • disabled：禁用元素
  • readonly：只读属性

input 对象方法

  • focus()：获得焦点(定位光标)
  • blur()：失去焦点的方法(移走光标)
  • select()：选中文本的方法

input 对象事件

  • onfocus：当获得焦点时
  • onblur：当失去焦点时

select、option 对象

使用 select、option 构成的二级联动表单

select 对象属性

  • optiions[]：设置或返回下拉列表中<option>标记构成的数组
  • selectedIndex：设置或<option>的索引号
  • length：指定下拉列表中<option>的个数
  • name：元素名称

option 对象属性

  • text：指<option>和</option>之间的文本
  • value：是指<option>标记的属性
