JavaScript ( 简称：”JS” )， 是一种小型、轻量级、面向对象、跨平台的客户端脚本语言。已经被广泛用于 Web 应用开发，常用来为网页添加各式各样的动态功能，为用户提供更流畅美观的浏览效果。通常 JavaScript 脚本是通过嵌入在HTML中来实现自身的功能的。

JavaScript程序可以在多种平台下运行，如：windows、linux、mac、ios等，且脚本程序只能在客户端的浏览器来运行，不能在服务器端来运行。

JavaScript 能干什么？

  • 表单验证：是JS最基本的功能(如下图)
  • 动态HTML：可以实现一些动态的、重复的效果
  • 交互式：人机交互，通过键盘或鼠标，与网页中的元素进行交互。

简介

名称的由来：最初是 网景公司(Netscape) 开发的 “LiveScript”，为自己的浏览器Navigator2.0开的客户端语言。想借助 Java 的名气很快成长起来，因此改名为 “JavaScript”。

Java 和 JavaScript 没有直接的联系。

变量

变量，是计算机语言中能储存计算结果或能表示值抽象概念。用于让你能够把程序中准备使用的每一段数据都赋给一个简短、易于记忆的名字，因此它们十分有用。

概念

变量本身是没有类型的，变量的类型来自于变量的值。

基本概念：

  • 变量是变化的一个量
  • 变量可以看成是一个”未知数”，x=10
  • 变量可以看成是一个”符号”、”代号”
  • 变量可以形象的看成是宾馆的”房间号”
  • 变量一般是指程序的数据
  • 变量是在内存中临时存在和运行的数据
  • 内存可以看成是一个个”小格子”，每个格子可以储存一个变量的名称和值

规则

命名规则：

  • 变量名可以包含字母、数字、下划线
  • 变量的名称一定要有意义
  • 变量名不能以数字开头，可以以字母或下划线开头。如：var 2name;(错误写法)
  • 变量名不能是系统关键字，如：var、switch、for、try、case、else、while等。
  • JavaScript 中的变量名是区分大小写的，如：name 和 Name 是两个变量
  • JavaScript 中每条语句一般以英文下的分号(;)结束，这个分号不是必须的，为了向 PHP 兼容最好加上分号
  • 如果变量名由多个单词构成的话，该如何表示？
    • “驼峰式命名”：第一个单词全小写，后面每一个单词首字母大写，如：var getUserName
    • “下划线命名”：所有单词全小写，中间用下划线连接，如：var get_user_name

类型

变量的数据类型，是指这个类型来源于”变量的值”，换句话说：值是什么类型，变量就是什么类型。

数据类型有：数值型、字符型、布尔型、未定义型(undefined)、空型(null)、数组(array)、对象(object)、函数(function)

变量的数据类型，分两大类：

  • 基本数据类型：数值型、字符型、布尔型、未定义型、空型(变量名只能存一个值)
  • 引用数据类型：数组、对象、函数(变量名可能存多个值)

条件判断、循环语句

if 条件判断

描述：条件成立，执行成立(true)下的代码，反之执行不成立(fasle)下的代码。

结构一：只判断真，条件为假时什么也不做

结构二：既判断真，也判断假

结构三：多条件判断

注意：虽然有多个条件，但各条件之间是”或”的关系。而且只能有一个条件成立，不能同时满足多个条件。

switch 分支语句

描述：根据一个变量的不同取值，来执行不同的代码。

结构说明：

  • switch、case、break、default都是系统关键字，都必须全小写
  • switch 后的小括号()：小括号内一般是一个变量名，这个变量可能有不同的取值
  • 每个 case 的值，与变量的值进行比对，如果一致就执行该 case 后的代码
  • 所有的 case 都是”或”的关系，只能有一个 case 会满足条件
  • 每个 case 中的代码执行完毕后，必须要用 break 语句结束，结束后程序将跳到 switch 结束大括号之后运行
  • 如果不写 break 语句，下边所有 case 语句都会执行一遍

if 多条件判断和 switc 的区别？
if 中的条件一般是一个范围(a > 100)，而 switch 中的条件一般是一个变量的不同取值。

描述：只要条件成立，就重复不断的执行循环体代码。

结构说明：

  • 在循环开始前，必须要对变量初始化(声明变量并给初始值)

  • 在循环体中，必须要有 “变量更新” 的语句，否则造成死循环

函数

函数，是将一段公共的代码进行封装，给它起个名字叫”函数”。它的作用是一次定义，多次调用，省去了代码的重写。例如：用户名的验证，验证码验证、邮箱验证、手机号码验证等. . .

例如：判断两个数的大小，只需封装代码结构之后调用

定义格式：

说明

  • function ：为系统关键字，全小写
  • 函数名称：创建函数时需要定义一个名称，命名规则与变量命名一样
  • ( )：是定义函数时需要接收数据的参数，按具体情况分析是否有参(可有可无)
  • 在参数中，不需要声明，直接使用
  • 在参数中，形参的名称与变量的命名规则一样。且形参不能用 “var” 关键字定义
    • 型参(形式参数)：参数里用形象的概念表示的参数，作用是来接收调用者传过来的数据，与变量相似
    • 实参(实际参数)：参数里实实在在的数据值，传递过来的真实数据
  • 函数功能代码：定义函数名称后紧跟着的是{ }中的函数代码
  • return：用于向函数调用者返回一个值，并立即结束函数的运行
    • return 用于终止函数的运行
    • break 用于终止各种循环的运行

变量

在函数中，变量分为：全局变量、局部变量。

  • 全局变量：可以在网页的任何地方(函数内、外部)使用的变量，称为”全局变量”
    • 在函数外部定义的变量，就是”全局变量”
    • “全局变量”既可以在函数外使用，也可以在函数内部使用
    • “全局变量”在网页关闭时，自动消失(释放空间)
  • 局部变量：只能在函数内部使用的变量，称为”局部变量”
    • “局部变量”在函数内部定义，在函数内部使用，则在函数外无法使用
    • “局部变量”在函数执行完毕就消失了

注意事项：

  • 在函数内部，省略关键字 var 定义的变量为 “全局变量”
  • var 关键字尽量别省略，省略后 “全局变量” 和 “局部变量” 无法区分

拷贝传值

拷贝传值：将一个变量的值”拷贝”一份，传给了另一个变量。两者没有任何联系，起到复制作用。
拷贝传值中，两个变量之间没有任何联系，修改其中一个变量的值，另一个不会改变。

基本数据类型存在于”栈内存”中，因为基本数据类型只有一个变量名和变量值。

引用传值

引用传值：将一个变量的数据地址”拷贝”一份，传给了另一个变量。两者是有联系的，起到
引用传值中，两个变量指向”同一个地址”，大家共享同一份数据，如果其中一个变量值发生改变则另一个变量值也发生改变。

引用数据类型存在于”栈内存”中，因为引用数据类型有多个值，将地址传入参数中。

引用数据类型的存储分两步：

  • 变量名和内存地址存在”快速内存” “栈内存” 中
  • 数据存在”满速内存” “堆内存” 中

匿名函数

没有名字的函数，不能单独定义也不能单独调用。
匿名函数一般是作为数据给其他变量赋值的，比如：普通变量、数组元素、对象属性赋值。

匿名函数的作用在于给实现一些功能比较小的代码进行封装，而不再重新命名定义函数，省去内存空间

二维数组的经常用在两层循环中，与二维表格的循环一模一样，即循环套循环(九九乘法表)。

二维数组的访问：数组名后跟多个连续的中括号[ ]。第一个括号代表第一维数组，以此类推。

对象

JavaScript 中的所有事物都是对象：字符串、数字、数组、日期，等等。对象是拥有”属性”和”方法”构成的数据。
对象是一种无序的集合数据类型，它由若干键值对组成。

基本概念

用形象的说法来描述：人就是一个”对象”，这个对象据有一些基本的特征比如身高、体重、姓名、性别、年龄、每个人可以有不同的本身(方法)如玩游戏、上班、开车等等. . .

内置对象

在 JavaScript 中系统提供的对象，不需要自己编写可以直接调用。

对象：

  • String：字符串对象，提供了对字符串进行操作的属性和方法
  • Array：数组对象，提供了数组操作的属性和方法
  • Date：日期对象，可以获取系统的日期时间信息
  • Boolean：布尔对象，一个布尔变量就是一个布尔对象(没有可用的属性和方法)
  • Number：数值对象，一个数值变量就是一个数值对象
  • Math：数学对象，提供了数学运算方面的属性和方法，如：绝对值、四舍五入

String属性、方法

一些常见的 String 对象的属性和方法。

  • length：获取字符串的长度，如：var len = str.length;
  • toLowerCase()：将字符串中的字母转成全小写，如：str.toLowerCase();
  • toUpperCase()：将字符串中的字母转成全大写，如：str.toUpperCase();
  • charAt(index)：返回指定下标位置的一个字符，如果没有找到则返回空字符串
    • 语法：str.charAt(index)
    • 参数：index 是指定的一个索引号，根据 index 索引号来返回一个字符

Demo：

  • indexOf()：返回一个子字符串在原始字符串中的索引值(查找顺序从左往右)，如果没有找到则返回 -1
    • 语法：str.indexOf(sub);
    • 参数：sub 代表要查找的一个子字符串

Demo：

  • lastIndexOf()：与 indexOf() 相似，区别在于查找顺序从右往左，如果没找到则返回 -1

    • 语法：str.lastIndexOf(str);
    • 参数：str代表要查找的子字符串

  • substr()：在原始字符串，返回一个子字符串

    • 语法：str.substr(startIndex[,length]);
    • 说明：从下标 startIndex 起，返回 length 个字符
    • 参数：startIndex 表示查找的起始索引号; length 可选，如果省略则返回几个字符

Demo：

  • substring()：与 substr() 一样，返回一个子字符串
    • 语法：str.substring(startIndex[, endIndex]);
    • 说明：返回从 startIndex 开始索引值到 endIndex 结束索引值之间的所有字符
    • 参数：startIndex 代表开始索引值; endIndex 可选项，如果省略则返回到结尾的所有字符

注意事项：如果省略第二个参数，那么 substr() 和 substring() 结果一样。

Demo：

  • split()：将一个字符串切割成若干段(返回一个数组)
    • 语法：str.split(分割号)
    • 参数：参数是一个分割号的字符串，用指定的分割号将字符串切成若干段

Demo：

Array属性、方法

一些常见的 Array 数组对象的属性和方法。
换句话说：一个数组变量，就是一个数组对象。

  • length：动态获取数组长度，如 var len = arr.length();
  • join：将一个数组转成字符串(返回一个字符串)，如 var str = arr.join(“,”);
    • 语法：str.join(连接号);
    • 说明：将一个数组，用指定的”连接号”连接成一个字符串
  • reverse：将数组中各元素颠倒顺序，如 var str.reverse();

数组元素的删除和添加：

  • Dlete 运算符：只能删除数组元素的值，而所占空间还在并且总长度不变(arr.length)
  • shift()：删除数组中第一个元素，返回删除的那个值，且所占空间也相对减 1
  • pop()：删除数组中最后一个元素，返回删除的那个值，且所占空间也相对减 1
  • unshift()：在数组前添加一个或多个数组元素，长度要改变，如：arr.unshift(“a”,”b”…);
  • push()：在数组结尾添加一个或多个数组元素，长度要改变，如：arr.unshift(“a”,”b”…);

Date属性、方法

Date 对象和 String 对象不太一样，定义了一字符串其实就是一个 String 对象，就可以直接调用其属性和方法。
Date 对象的使用，必须使用 new 关键字来创建，否则无法调用 Date 的属性和方法。

创建 Date 对象的方法

  • 创建当前日期对象，不带参数：
    • 实例：var today = new Date();
  • 创建指定时间戳日期对象，带参数(时间戳)：
    • 时间戳：是指某一个具体时间与当前时间之间的相差的毫秒值
    • 实例：var timer = new Date(10000);
  • 创建一个字符串的日期时间信息，带参数(字符串)
    • 实例：var timer = new Date(“2015/1/1 10:00:00”);
  • 指定多个数值参数：
    • var timer = new Date(2015,4,25);  //顺序为年、月、日、时、分、秒
  • getFullYear()：获取四位的年份
  • getMonth()：获取月份，取值为 0 ~ 11
  • getDate()：获取日期， 取值为 1 ~ 31
  • getHours()：获取小时
  • getMinutes()：获取分钟
  • getSeconds()：获取秒
  • getMilliseconds()：获取毫秒
  • getDay()：星期
  • getTime()：毫秒值，获取距离 1970 年 1 月 1 日至今的毫秒值
