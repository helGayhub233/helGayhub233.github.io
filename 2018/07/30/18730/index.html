CSS 全称为”层叠样式表”，主要是用于静态修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。目的是用于给HTML标记添加各种各样的表现，比如：文字样式、背景、文本样式等. . .

CSS 能够对网页中元素位置的排版进行像素级精确控制，支持几乎所有的字体字号样式，拥有对网页对象和模型样式编辑的能力。使用CSS必须以HTML为前提。

语法格式

语法结构：

注意事项：

  • CSS 的规则由 “选择器”和 “格式声明语句” 构成
  • 选择器：就是选择HTM标记，换句话说就是给哪个HTML标记加样式
  • 声明语句：由{ }构成，{ }中是各种格式语句
  • 属性名：就是 CSS 中的各种属性，这些属性名都是固定的
  • 属性值：一个属性名可取不同的值，这个值不加引号
  • 每一条格式语句，必须用英文下的分号 “；” 结束
  • CSS 中的数字单位都是 px（像素），这个 px 不能省略

基本选择器

选择器分为：基本选择器、组合选择器、伪类选择器三大类。

在基本选择器中，有一个比较常见的选择器，为”*”选择器：通配符里边的’星花’。
描述：它的作用是将匹配所有的 HTML 标记，所有的标记都会改变的。

Demo：

注意：” * “ 尽量少用，因为 IE6 以下版本不支持兼容性


标签选择器

描述：将匹配指定的 HTML 标记，比如指定 h1 标签。

Demo：

注意：CSS 标签选择器，与 HTML 标记的名称一样，但不能加尖括号。


类选择器

描述：给一类 HTML 标记加样式，这里所指的”一类”是：每个 HTML 标记都有一个 class 属性，且 class 的值一样。

在CSS样式中使用频率最高的就是类选择器（class）。

Demo：

注意事项：

  • class属性是公开属性，每个 HTML 标记都有
  • 类选择器的名称，必须以”.”开头，后限为 HTML 标记 class 属性的值，如：.box{color:red;}
  • HTML标记 class 属性值，不能以数字开头


id选择器

描述：给指定 id 的元素添加样式。与 class 相似，不同的是 id 选择器一般是配合js使用，不适合加样式。

Demo：

注意事项：

  • 网页中 HTML 标记的 id 属性的值，必须是唯一的
  • 每一个 HTML 标记都有一个id的公共属性
  • id 选择器的名称，必须以” # “开头，后跟 HTML 标记的 id 属性的值

组合选择器

描述：给多个元素加同一个样式，多个选择器至今用逗号 “,” 隔开，组合选择器又分为多元素选择器、后代元素选择器两类。

在层叠样式表中后代选择器普遍用的比较多。

伪类选择器

描述：给超链接加的样式，即链接的不同状态加样式。比如鼠标移动到链接时的样式等. . .
一个超链接，有四个状态：

  • 正常状态（:link）：鼠标没放上之前的链接的样式
  • 放上状态（:hover）：鼠标放到链接上时的样式
  • 激活状态（:active）：按住鼠标左键不松开的样式，这个状态特殊短暂
  • 访问过的状态（:visited）：按下鼠标左键，并弹起，这时的样式效果

注释

注释起到提示的作用，当查看某段代码时为了加以区分，应该加一段注释说明这些代码是干嘛的。
不同语言中注释的用法也不同。

HTML 的注释写法：

<—-HTML注释内容—->

CSS 的注释写法：

/*CSS 注释内容*/


属性样式

即层叠样式表的属性。


尺寸属性

与 HTML 的基本属性相同，基本上每个标记都有宽度、高度元素，除了一些比较特殊的单边标记例如：换行等. . .

属性：

  • width：元素宽度
  • height：元素高度


字体属性

层叠样式表当中声明字体的样式所用到的属性。

属性：

  • font-size：字体大小
  • font-family：字体样式，如：宋体、黑体、微软雅黑
  • font-style：斜体，取值：italic
  • font-weight：粗体，取值：bold


文本属性

给一段文本中添加不同的样式。

属性：

  • color：文本颜色
  • text-aligh：文本水平对齐方式，取值：left、center、right
  • line-height：行高，采用固定值和百分比：如12px、100%
  • letter-spacing：字间距
  • text-indent：首行缩进，如：12px
  • text-decoration：文本修饰线，取值：none(无)、underline(下划线)、overline(上划线)、line-through(删除线)


列表属性

给列表添加样式，例如去掉项目符号或编号前面的各种符号。

属性：

  • list-style：列表样式，取值：none

Demo：


边框属性

每个元素都可以加边框线。

属性：

  • border-left：左边框线
    • 格式：border-left：粗细 线形 线色
    • 线形：none(无线)、solid(实线)、dashed(虚线)、dotted(点状线)
    • Demo：border-left：5px dashed red;
  • border-right：右边框线
  • border-bottom：下边框线
  • border：同时给四个边加边框线


内边框属性

边框线到内容间的距离。
需要注意一点的是，平常我们所说的 width 和 height 属性，他们指内容的宽度和高度，不含内、外边距、边框线。

属性：

  • padding-left：左内填充距离，左边线到内容间的距离
  • padding-right：右内填充距离，与上方类似
  • padding-top：上内填充距离
  • padding-bottom：下内填充距离
  • 缩写形式：
    • padding：10px; //四个边的内填充分别为 10px
    • padding：10px 20px; //上下为 10px，左右为 20px
    • padding：5px 10px 20px; //上为 5px， 左右为 10px， 下为20px
    • padding：5px 10px 15px 20px; //顺序一定是 “上右下左”


外边框属性

边线往外的距离。

属性：

  • margin-left：左边框往外的距离
  • margin-right：右边线往外的距离
  • margin-top：上边线往外的距离
  • margin-bottom：下边线往外的距离
  • 缩写形式：
    • margin：10px; //四个边的内填充分别为 10px
    • margin：10px 20px; //上下为 10px，左右为 20px
    • margin：5px 10px 20px; //上为 5px， 左右为 10px， 下为20px
    • margin：5px 10px 15px 20px; //顺序一定是 “上右下左”


背景属性

顾名思义，给元素添加背景样式。

属性：

  • background-color：背景颜色
  • background-image：背景图片地址，如：background-image:url(images/bg.gif)
  • background-repeat：背景平铺方式，取值：no-repeat(不平铺)、repeat-x(水平方向)、repeat-y(垂直方向)
  • background-position：背景定位，如：background-position：水平方向 垂直方向
    • 用英文单词定位：background-position：left|center|right top|center|bottom;
    • 用固定值定位：background-position：50px 50px;
    • 用百分比定位：background-position：50% 50%;
    • 用混合定位：background-position：left 10px；
  • 缩写方式(组合)：
    • 方式：background：背景色 背景图 平铺方式 背景定位;
    • Demo：background: #ff0000 url(images/bg.gif) no-repeat center center


浮动定位

描述：浮动可以使块级元素并排显示，用于页面布局。
功能：可以实现多个块元素并列排版。

  • 浮动的元素，将向左或向右浮动，浮动到包围元素的边上，或上一个浮动元素的边上为止
  • 浮动的元素，不再占用空间，并且浮动的元素的层级要高于普通元素
  • 浮动的元素，一定是“块元素”，不管它原来是什么元素
  • 如果浮动的元素没有指定宽度，浮动后将尽可能的变窄，因此，浮动元素一般要定宽和高
  • 一行的多个元素，要浮动一起浮动

属性：

  • float：让元素浮动，取值：left、right
  • clear：清除浮动，取值：left(清除左浮动)、right(清除右浮动)、both(同时清除)

那么，如何让包围元素包住浮动元素?

你需要在浮动元素的下边，使用清除浮动的操作。

效果：


清除浮动

浮动和清除是一对的，当存在浮动时就有清除浮动。

清除浮动的功能：

  • 可以包围元素从 “视觉上” 包住浮动元素
  • 清除之下的其他元素将恢复默认排版
  • 有浮动，就得有清除
  • 如果包围元素指定了高度了，那么可以不使用清除功能


继承性

描述：外层元素的样式，会被内层元素进行继承，多个外层元素的样式，最终会”叠加”搭配内层元素上。

那么，什么样的CSS属性能被继承呢？

文本属性都会被继承：

color、font-size、font-family、font-style、font-weight
text-align、text-decoration、text-indent、letter-spacing、line-height

提示：<body>中的CSS属性，会被所有的子元素继承。


优先级

单个选择器的优先级：
越往下，优先级越低。如果被同名CSS属性的值，会被优先级高的选择器替换。

行内样式 > id选择器 > class选择器 > 标签选择器

多个选择器的优先级：
一般情况下，指向越准确，优先级越高。

.news h1 {color:red;} //优先级比较高
.titile{color:blue;} //优先级比较低


特殊属性

一些比较特殊的属性值。


display属性

功能：规则网页元素如何显示的问题。

取值：

  • none：将元素隐藏显示
  • block：以块元素显示，可以实现将行元素转成块元素
  • inline：以行内元素显示，可以实现将块元素转成行内元素


overflow属性

功能：当内容溢出时，溢出的内容该如何显示。

取值：

  • visible**：可见显示
  • hidden：隐藏显示
  • scroll：出现滚动条下拉显示
  • auto：自动判断显示方式


cursor属性

功能：改变网页中光标的类型。

取值：

  • help：帮助光标
  • wait：等待光标
  • hand：手型光标
  • 更多属性参考：w3school


定位

功能：调整元素的固定位置，不会跟着下拉栏移动。

取值：

  • position：元素定位方式
    • static：静态定位(默认状态不定位，一般不用)
    • fixed：固定定位(相对于浏览器窗口进行的定位且不占空间，层级高于普通元素，需要指定坐标，一定是块元素)
    • relative：相对定位(相对于本身的定位位置，占空间，层级高于普通元素，需要指定坐标)
    • absolute：绝对定位(相对于祖先定位元素定位，不占空间，层级高于平台元素，需要定位坐标，一定是块元素)

定位坐标：指定定位的元素，偏移目标元素多远的距离。

  • left：距离目标元素左边的距离
  • top：距离目标元素上边的距离
  • right：距离目标元素右边的距离
  • bottom：距离目标元素下边的距离
  • z-index：堆叠顺序，值越大越靠前排，取值：1 - 999

表格属性

常见的表格属性：

  • border-collapse：表格边框线合并，取值：collapse


盒子模型

我们可以把每一个HTML标记，都堪称是一个”盒子”。
这个”盒子”有哪些特征：内容的高度或宽度、边框线、内填充、外边距。

问题1：什么情况下？上下外边据会合并呢？
上下两个块元素，如果每一个元素都指定了四个外边距，那么上下相邻的那个外边距会发生合并现象，合并后取其中较大的外边距。

问题2：如果要实现上下两个<div>之间的距离为100px，该如何实现？
上下两个<div>中的其中一个指定margin-bottom：100px，而另一个<div>的margin-top：0px即可；或者也可以在上下<div>中间添加一个空的<div>，并给空<div>指定高度为100px。


HACK

针对不同浏览器，书写不同的CSS代码的过程，称为：”CSS HACK”。
也就是说：写一个 CSS 代码，让IE6识别，其他浏览器不识别。
下面针对不同浏览器，有几个符号：(这写符号是在 CSS 属性的前面加的，用于分辨不同的浏览器版本)

  • “星符号” IE6和IE7都识别。如：.box{*background-color:red;}
  • “ _ “ 只有IE6识别。如：.box{_background-color:green;}

注意：CSSHACK不是 W3C 的标准，因此我们尽量少用。如果需要调试兼容性则可以使用。
